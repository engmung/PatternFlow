# PatternFlow Manifesto

## 핵심 철학: 복잡성과 단순함의 이중주

### The Duality (이분법적 공존)

| 수학적 질서 (Logic) | 직관적 우연 (Intuition) |
|---------------------|------------------------|
| 규칙적인 알고리즘 | 예측 불가능한 우연 |
| 복잡함 | 단순함 |
| 무한성 | 유사성 |

> "이끌리는 것이 곧 나다"

**PatternFlow는 이 두 세계의 경계에 선다.**

---

## 프로세스: 혼돈에서 질서로

```
1. [COMPLEXITY] 수학적 알고리즘으로 복잡성 구축
       ↓
2. [SIMPLIFY]   픽셀화로 단순화 → 숨겨진 반복 패턴 발견
       ↓
3. [CURATE]     수많은 우연 중 "나에게 끌리는 것" 선택
```

## 최소한의 변화로 최대한의 효과

### 인터랙션 설계: 두 개의 패턴 비교

```
┌─────────────────────┐    ┌─────────────────────┐
│   Low Complexity    │    │   High Complexity   │
│   (작은 스케일)      │    │   (큰 스케일)        │
│                     │    │                     │
│   Wave Scale: 1.0   │    │   Wave Scale: 8.0   │
└─────────────────────┘    └─────────────────────┘
         │                          │
         └──────────┬───────────────┘
                    │
              [ ◉ 노브/슬라이더 ]
              값을 0.01씩 조절
                    │
                    ▼
     "복잡성이 클수록 작은 변화가 더 큰 파동을 만든다"
```

**사용자가 직접 체험:**
- 왼쪽(단순): 값을 바꿔도 변화가 미미함
- 오른쪽(복잡): 같은 값 변화에 드라마틱한 차이 발생
