# SMD Silkscreen Clarity

- Status: Open
- Source: Community review
- Target: PCB silkscreen / assembly

## Problem
R and C components have ambiguous silkscreen markings making assembly error-prone.

## Solution
- Add polarity marker (+) to capacitor silkscreen
- Use "R1", "R2" instead of generic rectangles for resistors
- Include value text (10k, 100nF) next to each footprint
- Increase silkscreen line width from 0.15mm to 0.2mm for readability
