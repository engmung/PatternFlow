Patternflow — Brand Assets
============================

Folders:
  og/         OG images (link previews) — 1200x630 + @2x + small
  svg/        Vector logo/mark
  favicon/    Browser favicons (16, 32, 48)
  app-icon/   iOS / Android / PWA (180, 192, 512, 1024)
  mark/      Generic mark PNGs (light/dark backgrounds)

File name convention:
  patternflow-<type>-<size>[-variant].<ext>
    og-1200x630-light.png   → 1200×630, light bg
    favicon-32.png          → 32×32 favicon
    app-icon-1024.png       → 1024×1024 app icon
    mark-512-on-dark.png    → 512×512, white mark on dark

Colors:
  Ink    #141414 / #0A0A0A
  Paper  #FAFAFA
